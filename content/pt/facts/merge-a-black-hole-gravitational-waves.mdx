## 🚨 Requisitos Obrigatórios (Deve ser rigorosamente seguido - Prioridade Máxima)

### Requisitos de Uso de Componentes
Cada artigo deve usar pelo menos uma vez todos os seguintes componentes:
- ✅ **Section** - Cada seção principal (obrigatório, geralmente 4-6 seções)
- ✅ **Callout** - Pelo menos 2-3 (usar diferentes tipos: dica/aviso/informação/sucesso)
  - **Importante**: Cada Section deve usar pelo menos uma vez um componente Callout
- ✅ **CardGrid/Card** - Pelo menos 1 grupo (3-4 cartões, mostrando comparação paralela)
- ✅ **Steps** - Pelo menos 1 fluxo de etapas (3-5 etapas)
- ✅ **Checklist** - Pelo menos 1 lista (3-5 itens)
- ✅ **FAQ** - Pelo menos 1 (3-4 conjuntos de perguntas e respostas)
- ✅ **Tabela** - Pelo menos 2-5

### Regras de Uso de Vídeos
- ⚠️ **O mesmo vídeo pode ser usado apenas uma vez**, não é permitido usar o mesmo vídeo várias vezes em um artigo
- Somente vídeos altamente relevantes para a palavra-chave principal devem ser incorporados (1-2 dos mais relevantes)
- Se todos os vídeos não forem relevantes, nenhum vídeo deve ser incorporado

### Regras de Uso do Componente Callout
- **Cada Section deve usar pelo menos uma vez um componente Callout**
- Usar diferentes tipos de tipo: dica, aviso, informação, sucesso
- Usado para destacar dicas importantes, avisos e conselhos profissionais
