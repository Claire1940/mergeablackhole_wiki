## 🚨 Requisitos Obrigatórios (Prioridade Máxima - Nível Máximo)

### Requisitos de Componentes
Cada artigo deve usar os seguintes componentes pelo menos uma vez:
- ✅ **Seção** - Cada seção principal (obrigatório, geralmente 4-6 seções)
- ✅ **Chamada de Atenção** - Pelo menos 2-3 (usando diferentes tipos: dica/aviso/informação/sucesso)
  - **Importante**: Cada Seção deve usar pelo menos uma vez um componente Chamada de Atenção
- ✅ **CardGrid/Card** - Pelo menos 1 grupo (3-4 cartões, mostrando comparações paralelas)
- ✅ **Passos** - Pelo menos 1 fluxo de passos (3-5 passos)
- ✅ **Lista de Verificação** - Pelo menos 1 (3-5 itens)
- ✅ **Perguntas Frequentes (FAQ)** - Pelo menos 1 (3-4 conjuntos de perguntas e respostas)
- ✅ **Tabela** - Pelo menos 2-5

### Regras de Uso de Vídeo
- ⚠️ **O mesmo vídeo pode ser usado apenas uma vez**, não é permitido usar o mesmo vídeo várias vezes em um artigo
- Somente vídeos altamente relevantes para a palavra-chave principal (1-2 dos mais relevantes)
- Se todos os vídeos não forem relevantes, não usar nenhum vídeo

### Regras de Uso do Componente Chamada de Atenção
- **Cada Seção deve usar pelo menos uma vez um componente Chamada de Atenção**
- Usar diferentes tipos: dica, aviso, informação, sucesso
- Usado para destacar dicas importantes, avisos e conselhos profissionais
