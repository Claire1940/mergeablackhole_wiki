## 🎮 Introdução à Cozinha de Buracos Negros
No jogo "Fusão de Buracos Negros", os jogadores embarcam em uma aventura emocionante para fundir buracos negros em entidades cada vez mais poderosas. Este guia o guiará pelo processo de cozinha de buracos negros, desde os fundamentos até as estratégias mais avançadas.

## 🌌 Entendendo a Fusão de Buracos Negros
### 🌟 O que é Cozinha de Buracos Negros?
A cozinha de buracos negros, ou fusão de buracos negros, é a mecânica central do jogo. Os jogadores começam com micro buracos negros e os fundem para criar buracos negros mais fortes, como poços de gravidade, grãos de vazio e muito mais.

### 🌟 Regras da Fusão de Buracos Negros
- **Regras Simples**: Funda um buraco negro com outro buraco negro para criar um mais forte.
- **Aumento de Potência**: Quanto mais buracos negros você fundir, mais forte se torna a entidade resultante.
- **Nível de Tier**: À medida que você funde buracos negros, o nível de tier aumenta, permitindo fusões ainda mais fortes.

## 🚀 Guia Passo a Passo para a Fusão de Buracos Negros
### 📝 Configuração Inicial
1. **Coletar Micro Buracos Negros**: Comece coletando o maior número possível de micro buracos negros.
2. **Aumentar o Nível da Base**: Aumente o nível da sua base para aumentar o limite de nascimento e o nível de tier.

### 📝 Progressão Central
1. **Fundir Micro Buracos Negros**: Funda micro buracos negros para criar poços de gravidade.
2. **Criar Poços de Gravidade**: Funda poços de gravidade para criar grãos de vazio.
3. **Construir Até Grãos de Vazio**: Continue fundindo para criar grãos de vazio.

### 📝 Táticas Avançadas
1. **Combinar Grãos de Vazio**: Funda grãos de vazio para criar esferas negras.
2. **Formar Esferas Negras**: Funda esferas negras para criar poços de gravidade.
3. **Criar Poços de Gravidade**: Funda poços de gravidade para criar núcleos de vazio.
